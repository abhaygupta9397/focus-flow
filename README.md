# Focus Flow

Focus Flow is a simple daily productivity app designed to help you plan and complete your most important focus tasks of the day.

The idea is straightforward: add 4 to 5 tasks you want to focus on, set a timer for each one, and work with a running countdown so you stay aware of time and build better focus momentum.

## Features

- Add daily focus tasks with a timer.
- Track each task using a countdown clock.
- Mark tasks as completed when the focus session is done.
- Get an audio alert when a task timer finishes.
- View the current date and time while planning your day.
- Switch between light and dark mode.

## Why This Project?

Focus Flow is built for everyday use. Instead of managing a long task list, it encourages you to choose a small number of meaningful tasks and complete them with intention.

Repeat the flow daily:

1. Add your focus tasks.
2. Start the timer.
3. Finish the task.
4. Move to the next one.

## Getting Started

Install the dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build the project for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
